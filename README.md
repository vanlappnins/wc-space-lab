# wc-space-lab

## 项目简介

`wc-space-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为页面资源的存储与展示空间。

## 目录结构

```
wc-space-lab/
├── pages/          # 存放各个独立 HTML 页面
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── docs/           # 相关文档或说明文件
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面，命名清晰，便于查阅。
- `assets/`：按类型或页面分类组织资源文件。
- `docs/`：存放与页面相关的说明、版本记录或使用指南。

## 页面归档说明

- 所有 HTML 页面均为独立、自包含的网页文件，可单独打开浏览。
- 页面内容可能包含实验性功能或静态展示，不保证长期可用性或兼容性。
- 归档的页面可能基于不同时间或目的创建，风格与功能各异。
